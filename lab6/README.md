# Album Browser (Lab 6 — Routing & HTTP)

## Description

This project is an Angular Single Page Application that displays albums and photos using routing and HTTP requests.
Data is fetched from the JSONPlaceholder REST API.

## Features

* Navigation between Home, About, Albums pages
* Album list view
* Album detail view with editing
* Delete album (simulated)
* Nested route for album photos
* Photo grid layout
* HTTP requests via Angular HttpClient
* Service layer (AlbumService)

## Technologies

* Angular
* TypeScript
* RxJS Observables
* Angular Router
* HttpClient

## How to run

1. Install dependencies:

```
npm install
```

2. Run the project:

```
ng serve
```

3. Open in browser:

```
http://localhost:4200
```

## API

Data is fetched from:
https://jsonplaceholder.typicode.com
